lazy loads

Lazy load, help to save load times and data usage avoiding downloading images that are out of the viewport.

The lazyLoad() function use 3 parameters

  1. animationDuration: the time to take the function for display the image, default = 300ms.
  2. windowPosition: the position that the omage have to be for start to beeing displayed, default = 0 (the image will be displayed when it top touch the bottom of the window)
  3. opacityLevel: the final opacity of the image, default = 1.0
